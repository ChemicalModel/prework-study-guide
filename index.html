HTML (Hypertext Markup Language)

  • Elements are a type of content or are used to divide and describes page.
  • Head elements contain hidden information about the webpage.
  • Body elements contain visibile content to the user.
  • HTML elements are closed in using angle brackets
  • Tags are what appears in the angle brackets
  • !DOCTYPE html: tells the browser to expect HTML on the page
  • html lang="[abbreviated language]": tells the browser the language for the webpage, "en" for example is english
  • title: defines the title for the webpage, helps with Search Engine Optimization
  • style: defines the CSS styles associated with a website
  • base: defines the base URL for a webpage
  • link: connects an external resource to the HTML document
  • meta: defines metadata such as character set, decription, keywords, author, and viewport
  • script: embeds client-side scripts in the HTML document
  • h1, h2, h3, h4, h5, h6: represent the level of heading a given text block represents, h1 being largest, h6 being smallest. Order matters
  • ul, ol, li: represent unordered, ordered and list items respectively, this website for example is ul which gives a bullet point list
  • img: contains information about images that are displayed on the webpage
  • src: is an attribute that defines the location of the image
  • alt: is an attribute that contains a text string that describes the appearance and functionality of the image
  • br: creates a line of empty space
  • a: stands for anchor and creates links to the same or other webpages
  • href="[link]": is an attribute that points to the URL for the link
  • section: semantic elements that define a section in a document that contains thematic content

CSS

  • A margin indicates how much space we want around the outside of an element.
  • A padding indicates how much space we want around the content inside of an element.
  • A selector defines the element or attributes to which the rules, or declarations, will apply
  • Declarations contain two important components: the CSS property we want to apply and the value of the property
  • A good bit of CSS is self-explanatory, width, height, color, background-color, etc. are all properties that can be set with values to style accordingly
  • Using a period before a word, such as "card" in this websites example, designates it a class selector. Any HTML element assigned to it will gain the property assignments from the CSS

Git

  • cd [file name]: changes what location or file you are viewing
  • code .: opens VS Code
  • touch [filename.filetype]: creates a file in the location you are viewing
  • git clone [url or SSH value]: clones repository from location or value entered
  • git add -A: adds all modifications in the current working branch
  • git status: checks what branch we are currently on
  • git checkout [branch]: switch to another branch and check it out into your working directory
  • git checkout -b branch-name: creates a new branch and switches to it
  • git commit -m "[descriptive message]": commit your staged content as a new commit snapshot
  • git pull origin [branch]: pulls changes from remote repository to the local environment
  • git push origin [branch]: pushes changes from local to remote repository

JavaScript

  • A variable is a named container that allows us to store data in our code.
  • Control flow is the order in which a computer executes code in a script.
  • Using console.log will output a message to the web console. In this website it is outputting the value of "topics"
  • Using conditional statements and loops can control the order in which code is executed.
  • An "if" statement is a way of introducing decision-making into the code, code will only execute if the statement meets a condition, or is true
  • When you have more than one condition, use "else if"
